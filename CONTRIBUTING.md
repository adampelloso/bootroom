# Contributing

This repo is structured for parallel work across multiple agents.  
Start with the Project Handbook.

## Start here
- Read `docs/project-handbook.md`.
- Then pick a workstream from `docs/workstreams.md`.

## Docs to consult
- Product scope: `spec.md`
- Project handbook: `docs/project-handbook.md`
- Insight roadmap: `docs/insight-market-map.md`
- UI reference: `docs/feed-ui-reference.md`

## Rules of engagement
- Keep UI language neutral (no betting terms).
- Avoid touching files owned by another stream.
- Update the relevant docs if you change behavior or templates.

## PR expectations
- Short summary of changes + rationale.
- Note any assumptions or data limitations.
